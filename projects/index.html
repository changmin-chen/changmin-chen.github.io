CT

Organ Segmentation/ 3D Modeling

In this project, I designed and develop the image processing pipeline to build 3D human organ model from the subject's computed tomography (CT) image. By using Python, I processed the abdomen CT images. The images were in common, standard medical image format, such as DICOM and NIFTI. By using Pytorch, I trained the machine learning (ML) model. The ML model will automatically segment the human organs from abdomen CT images. I evaluated the ML model performance for segmentation after the model was trained. By using VTK, I eventually build the 3D model of human organ. This 3D model will help doctor and physicians visualize the patient's organ stereotactically, catching the spatial information of the organs in a more straightforward way.
